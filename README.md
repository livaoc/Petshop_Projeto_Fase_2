# Projeto de desenvolvimento de um sistema WEB para um Petshop - Fase 2
Fase 2 do Projeto da disciplina Fundamentos de Sistemas Web: desenvolver um sistema web para um petshop, utilizando javascript, CSS e boostrap.

### Funcionalidades e Ajustes Realizados comparado à Fase 1
* **Home:** Foi adicionado carrossel promocional (Bootstrap) e mensagem de saudação baseada no horário do usuário utilizando JavaScript diretamente no HTML.
* **Serviço:** Foi implementado um formulário completo para o cadastro do cliente e do pet, incluindo o agendimento de serviço com opção de tele-busca ou não.
* **Carrinho:** Foi implementado um script em JavaScript para calcular o valor total da compra e remover itens dinamicamente.
* **Acessibilidade:** Inclusão do atributo 'alt' nas imagens para leitores de tela e campos dentro do formulário.
